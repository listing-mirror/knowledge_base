Getting Started | Creating Listings

Learn how to create new listings on Listing Mirror, and set expectations for our accepted workflow

We don't run an import for non-Amazon or non-WooCommerce marketplaces. New products need to be created on Listing Mirror. Listing Mirror cannot manage anything created off-platform.

Creating New Products

  1. From the Manage Listings page, click Add New Product on the top right of the page.
  2. Input all required information, and click Confirm.
  3. Search for the created SKU, and click on its Product Catalogue.
  4. Input all required information, and click Save Changes.

To add variations, use the Actions dropdown on the top of the page, and click Add Variant.

Once satisfied, you can create market listings using the Market Listings box.
