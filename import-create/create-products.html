Getting Started | Create Products

Learn how to create new products, and set expectations for our accepted workflow

We don't run an import for non-Amazon or non-WooCommerce marketplaces. New products need to be created on Listing Mirror. We cannot manage anything created off-platform.


Create New Products

  1. From the Manage Listings page, click Add New Product on the top right of the page.
  2. Input all required information, and click Confirm.
  3. Search for the created SKU, and click on its Product Catalog.
  4. Input all required information, and click Save Changes.

To add variations, use the Actions dropdown on the top of the page, and click Add Variant. Learn more

Once satisfied, you can create listings using the Market Listings box.
