If you need to link a listing to a different product

Learn how to link, or unlink a listing from a product in Listing Mirror

To maintain accurate product data and inventory, it is essential to link your listings to the appropriate products. If you have identified multiple products for the same listing, it's crucial to link the listing to the correct product.

A Product is defined as any item on the left side of the Manage Listings page. A Listing is defined as any item on the right side of the Manage Listings page.

It's common to come across two products for the same listing when you use different SKUs on various markets. In the below example, a user uses SKU HI-1090 on Amazon, but SKU HI-1090-E on eBay.

Link a listing to the correct product, creating one product

  1. From the Manage Listings page, search for the primary product, and click on the title.
  2. Using the Actions tab, click Link Listings.
  3. Search for the listing you need to link to the primary product.

In the example below, you can see the inventory levels are properly sent to both market listings.

Unlink a listing from a product, creating two products

  1. From the Manage Listings page, search for the primary product, and click on the title.
  2. Using the Actions tab, click Unlink Listings.
  3. On the page, create a new Listing Mirror SKU (LMSKU), and select a radio button under the marketplace you want to assign it to.
