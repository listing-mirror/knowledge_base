Getting Started | FBA, WFS, and 3PL

Learn how Fulfilled By Amazon, Walmart Fulfillment Services, and other third-party logistics operate in Listing Mirror

Listing Mirror has limited control over your third-party inventory sources. Instead, these sources send us inventory levels, allowing us to fill orders with them.


Multi-Channel Fulfillment

Using this third-party inventory, you can sell it on any platform. This is know as Multi-Channel Fulfillment. We provide you with the tools to sell each source on any market of your choosing.

  1. From the menubar, click Integrations, followed by Manage Integrations.
  2. Select a marketplace.
  3. From the Inventory tab, observe the green and red boxes.
  4. Adjust which source you'd like to use, in the order you'd like to use them.

The order of the inventory source will determine which inventory we sell first.
