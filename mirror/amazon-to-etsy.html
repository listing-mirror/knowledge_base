Getting Started | Amazon to Etsy

Learn how to move the data from your Amazon listings to new Etsy listings, avoiding common pitfalls

One of our pillars is listing creation. Using your Product Catalog data, we build your Etsy listings with ease.

Before getting started, you'll need to ensure your Etsy account is properly set up. Failing to do so can result in your listings not going live.


Common Pitfall: Fixed Price Shipping Profile

Before you can add listings to Etsy, you need to create a Shipping Profile. If you're already an Etsy seller, you may have one. If not, you'll need to create one.

  1. On Etsy, open your Account Settings.
  2. Click Shop Manager, then click Settings, followed by Shipping Settings.
  3. Click Add A Shipping Profile.
  4. Fill out all required fields.

Creating Listings

Create Individual Listings

  1. From the Manage Listings page, find a product missing from Etsy.
  2. Click the blue Add to Marketplace button.

If there are no issues, and you're syncing, the listing will go live.

Create Multiple Listings

  1. From the Manage Listings page, find a group of missing products on Etsy.
  2. On the left, click the checkbox for each listing you'd like to create.
  3. Using the dropdown, click Create.
  4. Click the Etsy checkbox, then hit Confirm.

Depending on the amount of listings created, it may take up to a minute to display them all.

Create Multiple Listings Using Templates

  1. From the menu bar, click Listings, followed by Bulk Edit Templates.
  2. From the dropdown, select Create and Delete, followed by Download Template.
  3. Once downloaded, open the sheet and input C in the marketplace column, and SKU row for the desired product.
  4. Save the sheet, head back to the Bulk Edit Templates, and click Upload Template.

C is short for Create. You can review the full documentation on our knowledge base. Learn more

Need to view the status of your bulk template upload? Review your Bulk Upload History page.
