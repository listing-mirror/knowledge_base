Getting Started | Category Mapping

Learn how to map Amazon categories you use to be similar to other markets

After an Amazon listing is in Listing Mirror, we'll map its assigned category to the most relevant category on another marketplace. You're able to change these at any point from the Category Mapping page. Learn more

Please note, these will only apply to new listings. No existing listings will be changed.


Default Category Mapping

If there's a category you want your products assigned to, you can select a default for each marketplace. Any product that doesn't have a specific mapping will be assigned this category.

Default category mapping for an example client who only sells dog products.


Amazon To Market Specific Mapping

Since we map these categories for you, you may want to change them in the future. You're provided with every category possibility you have on Amazon. Click Override, and select the desired category.

Specific category mapping for someone who usually sells dog products, but found some bird products to sell.
