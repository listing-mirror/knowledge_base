Learn how to edit the core listing data in bulk using this template

The Market Listing Core Edit template allows you to edit the data on the Listings tab for a given marketplace. Each market allows you to change specific fields, and if you need to make these changes, this is the template you'll want to use. 

This template is great for updating fields, such as:

  • Etsy's  Who Made It field
  • eBay's Shipping, Payment, & Return Profile ID fields
  • Shopify's  Handle field
  • Newegg's  Package Quantity field

While you're able to update universal fields such as Bullet Points, Titles, Descriptions, etc., this will only update the marketplace of your choice. They will no longer be uniform across your stores. 

As a bonus, this template can help you identify errors in bulk, allowing you to view and make changes based on what error is given. See Editing Templates > System Warnings below. 

Downloading Templates

To start, you'll want to go to the  Bulk Edit Templates page. You can find that under  Listings on your menubar. From the dropdown, select  Market Listing Core Edit, and click  Download. This template will allow you to pick a specific marketplace, so you're able to refine your results. 

 

 

Editing Templates

Listings Tab

This is where you'll make your content changes. Below, you'll find what each portion of the sheet means, and what happens when you edit these cells. 

Main Data  (Column A - AM)

This row contains information that will update both the parent listing, and your variations for that listing. These cells include: title, description, variation names, and more. 

Variation Data (Column AN - BE)

This row contains information that will update variation specific data. These fields include: UPC, MPN, variation values, shipping information, and more.

System Warnings (Column BF - BK)

This row contains information related to errors and warnings you may have on a given SKU. This is perfect if you have a lot of errors that you need to address, but don't want to view each listing individually to resolve them. 

Blue Cells

If a cell is blue, it's being controlled by the Product or Variant page for that listing. Making changes to these cells will remove that connection, allowing the data to be marketplace specific, rather than uniform. 

Grey Cells

If a cell is grey, the cells are being used as filler. These are used as spacers, and should not be edited. Including data in these cells will result in an error while uploading. 

Yellow Cells

If a cell is yellow, it's required by the marketplace. Keep in mind, other fields might be required by the marketplace based on what category you select, but these fields will always be required. 

 

Uploading Templates

Finally, once you've completed editing your template, you'll want to head back to the  Bulk Edit Templates page and upload your template. Select  Market Listing Core Edit from the dropdown, then simply hit  Upload.

 

You can check the progress of the upload on the  Bulk Upload History page, under  reports on the menubar. If there are any errors, you can read about them on your  Notification History page.
