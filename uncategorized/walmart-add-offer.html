Listing Creation | Add Walmart Offers

Learn how to add an offer to Walmart listings using a pre-existing barcode, foregoing the need to input details manually on Listing Mirror

Similar to Amazon, you're able to use existing listings to start selling items on Walmart with minimal effort. During listing creation, we'll attempt to match your barcode to an existing Walmart listing, and fill in required fields for you.


Add Offer

To start, ensure your Product Catalog has a barcode in the Identifiers box. To add barcodes in bulk, use the Product Edit template. Learn more

Individual Listings

  1. From the Manage Listings page, search for a desired SKU.
  2. Click the plus button under the Walmart column.
  3. Observe the Building symbol, designated by a faded clock.

This process can take up to 24 hours, but most see results within the hour. If a listing exists using this barcode, the listing won't be editable, but will be live. To make changes, click Convert To Editable. Learn more

If no listings exist, you'll need to input any required details to create the listing on Walmart.
