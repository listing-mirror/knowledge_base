If you would like to change the details of all your products, now you can edit your Product Catalog in bulk. An exciting feature of this spreadsheet is that you can now set your data to be the same as the Product Catalog across all integrations!

Where to find the template

Going to our  Bulk Edit Templates page, you can download the Product Edit template:

 

New features

The Product Edit spreadsheet is a little different from the other ones we have: 

1. Gray columns : Gray columns are ones that should not be filled in. If you fill these out, you will have an error while uploading the spreadsheet. We put in those grey columns because child products don't have some fields, like brand, description or item_length. 

 

2.  Actions : On the spreadsheet, there is a third sheet called  Actions. Using this sheet, you can have the same details, like title or description, across all channel integrations.

You can do this by changing your input in the  Overwrite on All Integrations column on the  Actions sheet.

 

The default value in this column is  F, meaning "false".

Changing the value to T (for "true") will erase and push the data to be the same as the Product Catalog across all channel integrations. 

3.  Do not change the  listing_id  and  parent_listing_id  columns  on the first page of the spreadsheet . These fields are identification numbers for our internal use only. Changing these fields will give you an error when you upload the template. 

Template Overview

On the first page of the spreadsheet, ProductEdit, you can fill in all the fields that you would normally enter in your Listing Mirror account. 

The second page of the spreadsheet is the Lookup page, for internal purposes. Please do not delete or edit this page. If you do, you will get an error when you upload the template. 

The third page is Actions, as explained in New features. 

If you would prefer to change details in your Listing Mirror account and not on a spreadsheet, you can do that in the Product page on Listing Mirror directly.
