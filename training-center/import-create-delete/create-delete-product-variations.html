Getting Started | Create & Delete Product Variations

Learn how to create new and delete existing product variations on Listing Mirror

It's important to note that we currently only support automatic import for Amazon and WooCommerce listings. Any listings from other markets will need to be manually created on our platform. We are unable to manage any listings created outside our platform.

Create product variations for existing products

Individual Listings

  1. On a product page, click the Actions dropdown, followed by Add Variation.
  2. Input any required details, and click Create.

Multiple Listings

  1. From the menu bar, click Reports, followed by Import / Export.
  2. Using the dropdown, select Product Add, followed by Download Template.
  3. Open the spreadsheet (.xlsx), and input all the desired information.
  4. Save changes, head back to Import / Export, and click Upload Template.

Delete a product variation that's no longer needed

If you're trying to delete a listing, you don't need to delete the product. A product can exist with no consequence, even if no market listing is associated with it.

Individual Products

  1. From the Manage Listings page, search for a desired SKU, and click the product title.
  2. In the Variations box, click the delete button.
  3. Save changes.

This will only delete the product. You'll still have to delete the market listing. Learn more

Multiple Listings Using Spreadsheets

  1. From the menu bar, click Listings, followed by Bulk Edit Templates.
  2. Using the dropdown, select Create and Delete, followed by Download Template.
  3. Once downloaded, open the sheet and input X in the Remove Entirely column for the desired SKUs.
  4. Save the sheet, head back to the Bulk Edit Templates, and click Upload Template.

You can review the full documentation on our knowledge base. Learn more (coming soon)
