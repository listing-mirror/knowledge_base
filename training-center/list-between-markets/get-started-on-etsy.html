Get started listing on Etsy

Learn how to use the data from your other market listings to create new Etsy listings, avoiding common pitfalls

To begin, it is essential to ensure that your Etsy seller account and Etsy integration settings are correctly configured. Neglecting to do so may prevent your listings from going live.

Common Pitfall: Create and assign a shipping profile

To add listings to Etsy, it is necessary to create a Shipping Profile. If you are already an Etsy seller, you may already have one. However, if you do not have a Shipping Profile, you will need to create one.

  1. On Etsy, open your Account Settings.
  2. Click Shop Manager, then click Settings, followed by Shipping Settings.
  3. Click Create Profile.
  4. Fill out all required fields.

Common Pitfall: Create and assign a return policy

Just like the Shipping Profile, you will also need to have a Return Policy in place.

  1. On Etsy, open your Account Settings.
  2. Click Shop Manager, then click Settings, followed by Return Policy.
  3. Click Create Policy.
  4. Fill out all required fields.

Set Default Policies

  1. In Listing Mirror, from the menu bar, click Integrations, followed by Manage Integrations.
  2. Click any Etsy channel, followed by the Advanced tab.
  3. Under Default Shipping Template or Default Return Profile, select an option.

This will only affect future listings, and won't change existing ones.

Set Individual Policies

  1. From the Manage Listings page, search for your listing.
  2. Click the Etsy Market Listing for the desired listing.
  3. On the Listings tab, locate the Shipping Template ID or Return Profile ID field, and select an option.

List your products to Etsy

Create Individual Listings

  1. From the Manage Listings page, find a product missing from Etsy.
  2. Click the blue Add to Marketplace button.

Create Multiple Listings

  1. From the Manage Listings page, find a group of missing products on Etsy.
  2. On the left, click the checkbox for each listing you'd like to create.
  3. Using the dropdown, click Create.
  4. Click the Etsy checkbox, then hit Confirm.

Create Multiple Listings Using Spreadsheets

  1. From the menu bar, click Reports, followed by Import / Export.
  2. From the dropdown, select Create & Delete, followed by Download Template.
  3. Once downloaded, open the sheet and input C in the marketplace column and SKU row for the desired product.
  4. Save the sheet, head back to the Import / Export, and click Upload Template.
