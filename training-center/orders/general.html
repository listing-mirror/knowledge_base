Getting Started | General Orders

Learn how to use the general inventory features built into Listing Mirror

We provide features that allow you to control how orders are fulfilled, as well as edit the data provided by your customer.


Manual Orders

  1. From the menu bar, click Orders, followed by Show All Orders.
  2. From the Actions dropdown, click Create Manual Order.
  3. Input all required information.

Once created, you can then fill the order yourself, or use third-party logistics inventory.


Split Orders

If an order has multiple products from different inventory sources, you can split the order and fulfill them using different third-party logistics sources.

Please note, the market will not be notified until ALL split orders have shipped.

  1. From the menu bar, click Orders, followed by Show All Orders.
  2. Search for an order, and click on the order number.
  3. Using the Actions dropdown, click Split Order.
  4. Input the quantity you want to split, followed by Split Order.

If you'd like to automate this process, you can enable Auto-Split Fulfillment in your integration settings. Learn more


Update Order Details

If your customer contacts you, or you notice they input invalid information, you can manually edit their delivery information.

  1. From the menu bar, click Orders, followed by Show All Orders.
  2. Search for a desired order number.
  3. Using the Actions dropdown, click Update Order Details
  4. Fill out or update any required details, and click Save Details.
