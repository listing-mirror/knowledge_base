Getting Started | Third-Party Fulfillment Orders

Learn how to view, manage, and edit orders made using inventory from a third-party on Listing Mirror

When your customers place orders using your third-party logistics (3PL) inventory, we'll send those orders to the appropriate place. The 3PL will fill the order, give us the tracking number, and we'll notify the marketplace that the order has shipped.


Autofill MCF, DLV, and RSL orders

If you want to avoid sending orders manually to Amazon, Deliverr, or ShipWorks you can enable Autofill MCF / DLV / RSL Orders on an integration basis. "MCF" is short for Multi-Channel Fulfillment, "DLV" is short for Deliverr, and "RSL" is short for ShipWorks.

  1. From the menu bar, click Integrations, followed by Manage Integrations.
  2. Select a marketplace.
  3. From the Inventory tab, enable Autofill MCF Orders, Autofill DLV Orders or Autofill RSL Orders.

Blocking Amazon Logistics & using Amazon Blank Box

When using Fulfilled By Amazon (FBA) inventory, Amazon gives you the option to send your customers unbranded tracking information, and shipping boxes.

This is required for Walmart.

  1. From the menu bar, click Integrations, followed by Manage Integrations.
  2. Select a marketplace.
  3. From the Inventory tab, enable Block Amazon Logistics and/or Amazon Blank Box.

Auto-split orders containing MF and 3PL items

If you want to avoid splitting orders manually that contain 3PL & merchant fulfilled (MF) inventory, you can enable the Auto-Split Fulfillment option on an integration basis.

  1. From the menu bar, click Integrations, followed by Manage Integrations.
  2. Select a marketplace.
  3. From the Inventory tab, enable Auto-Split Fulfillment.

Cancel orders sent to 3PL platforms

  1. From the menu bar, click Orders, followed by Show All Orders.
  2. Search for an order.
  3. Under Fulfillment Actions, click Cancel MCF.

From there, you'll want to follow the standard process for cancelling MF orders. Learn more

If you're unable to cancel in time, you may need to cancel the order on the respective 3PL platform.


Handling Amazon MCF returns

To manage an MCF return via Amazon, you'll need to create a custom return. You can choose to send the item back to Amazon, or to another location.

Create Returns

  1. From your Amazon seller account, go to Orders, followed by Manage Orders.
  2. Click Advanced Search.
  3. In the Search dropdown list, ensure Order ID is selected.
  4. From the Manage Orders page, click the Order ID.
  5. From the Order Details page, click the Create Customer Return tab.
  6. For each item, select Return Quantity, and Return Reason.
  7. Click Submit to generate the RMA and return shipping label.
  8. Send the RMA and shipping label to your customer.

Track Returns

  1. Follow steps one through five above.
  2. From the Order Details page, click the Return Status tab.

Once the returned package is processed at the fulfillment center, the return status will show Processed. If the item has additional information, you can find that in the Disposition column.


Definitions for each order status

On the orders page, we'll display the status of your orders.

  • Won't Fill: The order won't or can't be filled by third-party logistics.
  • Awaiting Fulfillment: The order is waiting to be filled by third-party logistics.
  • Filled, Awaiting Tracking: The order has been filled by third-party logistics, and we're waiting for tracking data.
  • Partially Filled: Parts of the order have been provided tracking details by third-party logistics, but not all.
  • Fully Filled: The order has been filled by third-party logistics, and they've provided us with tracking details. We've successfully notified the market that the order has shipped.
