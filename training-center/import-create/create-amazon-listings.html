Getting Started | Create Amazon Listings

Learn how to create Amazon listings while using Listing Mirror, and set expectations for our accepted workflow

We provide multiple ways for you to add new Amazon products. If you need to add offers to existing listings, or create new listings, we have options for you.


Add Offers To Existing ASINs

Amazon sellers can add offers to existing products that another seller owns.

Individual Listings

  1. From the Manage Listings page, click any Amazon store.
  2. On the top left of the page, click Amazon Actions.
  3. Using the dropdown, click Add New Listing.
  4. On the page, input all required information. Confirm this is the correct product at the bottom of the page.
  5. Click Add Amazon Listing.

Multiple Listings

  1. From the menu bar, click Listings, followed by Bulk Edit Templates.
  2. Using the dropdown, select Amazon Add Offer and an Amazon integration, followed by Download Template.
  3. Open the sheet and input all required information.
  4. Save changes, head back to Bulk Edit Templates, and click Upload Template.

Required fields are, Product Identifier Type, Product Identifier, SKU, Price, Quantity, and Condition.

Please allow a few minutes for your listing to appear. Amazon has to review your request, and accept your submission. If they decline your request, we'll notify you in your notification center.

Would you like to use the data from an Amazon listing you don't have permission to sell on another marketplace? Consider using an untracked listing! Learn more


Add Brand-New Products

Flat file are something Amazon provides to its sellers. Uploading a flat file using Listing Mirror allows us to build a Product Catalog while Amazon processes the data. Learn more about flat files

  1. From the Manage Listings page, click any Amazon store.
  2. On the top left of the page, click Amazon Actions.
  3. Using the dropdown, click Create From Inventory File Template.
  4. Read the information on the page before uploading.
  5. Click the upload window, and select your spreadsheet.

Add Products Directly On Amazon

If you're more comfortable using Amazon's website, you can continue to add products to Amazon. We run an import every two hours to bring in any new Amazon products.
