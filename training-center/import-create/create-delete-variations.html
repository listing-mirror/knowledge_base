Getting Started | Create & Delete Product Variations

Learn how to create and delete variations for your products, and set expectations for Listing Mirror's accepted workflow

We don't run an import for non-Amazon or non-WooCommerce marketplaces. New products need to be created on our platform. We cannot manage anything created off-platform.


Create product variations for existing products

Individual Listings

  1. From any Product Catalog, click the Actions dropdown, followed by Add Variation.
  2. Input any required details, and click Create.

Multiple Listings

  1. From the menu bar, click Listings, followed by Bulk Edit Templates.
  2. Using the dropdown, select Product Add, followed by Download Template.
  3. Open the sheet and input all the desired information.
  4. Save changes, head back to Bulk Edit Templates, and click Upload Template.

To create variations, start by filling out the parent fields in column A to AC. Then, fill out the child fields AD to CC. Parent details only need to be filled out once. For each variation, only fill out the child information. Need to see an example? Learn more

You can create listings on the Manage Listings page, or use a Create & Delete template. Learn more


Delete product variations for existing products

Individual Products

  1. From the Manage Listings page, search for a desired SKU, and click the parent Product Catalog.
  2. In the Variations box, click the delete button.
  3. Save changes.

Multiple Listings Using Templates

  1. From the menu bar, click Listings, followed by Bulk Edit Templates.
  2. Using the dropdown, select Create and Delete, followed by Download Template.
  3. Once downloaded, open the sheet and input X in the Remove Entirely column for the desired SKUs.
  4. Save the sheet, head back to the Bulk Edit Templates, and click Upload Template.

You can review the full documentation on our knowledge base. Learn more

Deleting a Product Catalog will also delete the associated Market Listings.
