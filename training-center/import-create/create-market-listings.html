Getting Started | Create Market Listings

Learn how to create your market listings once your product data has been filled out on Listing Mirror

We don't run an import for non-Amazon or non-WooCommerce marketplaces. New products need to be created on our platform. We cannot manage anything created off-platform.


Create Market Listings

If you're getting started with Listing Mirror, and want to learn how to crosspost data from another channel, continue to our crossposting guide.

Individual Listings

  1. From the Manage Listings page, search for a desired SKU.
  2. On the right, click a plus button to create the market listing.

Please note, if a listing is a variation, we'll create all children associated with the parent.

Multiple Listings

  1. From the Manage Listings page, select a group of SKUs.
  2. Using the Actions dropdown, select Create.
  3. Select your desired marketplaces for creation, followed by Confirm.

Create Multiple Listings Using Templates

  1. From the menu bar, click Listings, followed by Bulk Edit Templates.
  2. From the dropdown, select Create and Delete, followed by Download Template.
  3. Once downloaded, open the sheet and input C in the marketplace column, and SKU row for the desired product.
  4. Save the sheet, head back to the Bulk Edit Templates, and click Upload Template.

C is short for Create. You can review the full documentation on our knowledge base. Learn more

Need to view the status of your bulk template upload? Review your Bulk Upload History page.
