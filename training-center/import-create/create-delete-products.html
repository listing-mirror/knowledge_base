Getting Started | Create & Delete Products

Learn how to create and delete your existing products on Listing Mirror

We don't automatically import listings from any market aside from Amazon and WooCommerce after you've added the integration. New products or listings need to be created using our software, as we cannot manage anything off platform.

Create a product for your listing data

Individual Listings

  1. From the Manage Listings page, click Add New Product on the top right of the page.
  2. Input all required information, and click Confirm.
  3. Back on the Manage Listings page, search for the created SKU, and click the product title.
  4. Input all required information, and click Save Changes.

Multiple Listings

  1. From the menu bar, click Listings, followed by Bulk Edit Templates.
  2. Using the dropdown, select Product Add, followed by Download Template.
  3. Open the spreadsheet (.xlsx), and input all the desired information.
  4. Save changes, head back to Bulk Edit Templates, and click Upload Template.

You can review the full documentation on our knowledge base.

Delete a product that's no longer needed

If you're trying to delete a listing, you don't need to delete the product. A product can exist with no consequence, even if no market listing is associated with it.

Multiple Listings

  1. From the Manage Listings page, select a group of SKUs.
  2. Using the Actions dropdown, select Delete.
  3. Click Select All Channels, followed by Confirm.

Multiple Listings Using Spreadsheets

  1. From the menu bar, click Listings, followed by Bulk Edit Templates.
  2. Using the dropdown, select Create and Delete, followed by Download Template.
  3. Once downloaded, open the sheet and input X in the Remove Entirely column for the desired SKUs.
  4. Save the sheet, head back to the Bulk Edit Templates, and click Upload Template.

You can review the full documentation on our knowledge base.

Deleting a product will also delete the associated Market Listing.
