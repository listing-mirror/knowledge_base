Getting Started | Multi-Channel Fulfillment

Learn how to view, manage, and edit orders made using your 3PL inventory

When your customers place orders using your 3PL inventory, we'll send those orders to the appropriate 3PL, where they'll be filled, and tracking data will be sent back by said 3PL.


Cancelling Orders

  1. From the menubar, click Orders, followed by Show All Orders.
  2. Search for an order.
  3. Under Fulfillment Actions, click Cancel MCF.

From there, you'll want to follow the standard process for cancelling merchant fulfilled orders. Learn more

If you're unable to cancel in time, you may need to cancel the order on the respective 3PL's platform.


Returns

To manage an MCF return via Amazon, you'll need to create a custom return. You can choose to send the item back to Amazon, or to another location.

Creating Returns

  1. From your Amazon seller account, go to Orders, followed by Manage Orders.
  2. Click Advanced Search.
  3. In the Search dropdown list, ensure Order ID is selected.
  4. From the Manage Orders page, click the Order ID.
  5. From the Order Details page, click the Create Customer Return tab.
  6. For each item, select Return Quantity, and Return Reason.
  7. Click Submit to generate the RMA and return shipping label.
  8. Send the RMA and shipping label to your customer.

Tracking Returns

  1. Follow steps one through five above.
  2. From the Order Details page, click the Return Status tab.

Once the returned package is processed at the fulfillment center, the return status will show Processed. If the item has additional information, you can find that in the Disposition column.


Splitting Orders

If you don't want to split orders manually that contain Multi-Channel Fulfillment & Merchant Fulfilled order, you can enable the Auto-Split Fulfillment option on an integration basis

  1. From the menubar, click Integrations, followed by Manage Integrations.
  2. Select a marketplace.
  3. From the Inventory tab, enable Auto-Split Fulfillment.

Order Status

On the orders page, we'll display the status of your orders.

  • Won't Fill: The order won't or can't be filled by the 3PL.
  • Awaiting Fulfillment: The order is waiting to be filled by the 3PL.
  • Failled, Awaiting Tracking: The order has been filled by the 3PL, and we're waiting for tracking data.
  • Partially Filled: Parts of the order have been provided tracking details by the 3PL, but not all.
  • Fully Filled: The order has been filled by the 3PL, and they've provided us with tracking details. We've successfully notified the market the order has shipped.
